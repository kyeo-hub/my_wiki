# 欢迎来到我的知识库

这是一个使用 Hugo 和 Docdock 主题构建的个人知识库。

## 快速开始

- 左侧导航栏浏览所有文档
- 使用搜索功能查找内容
- 支持 Markdown 语法编写文档

## 目录结构

- **入门指南** - 基础知识和介绍
- **技术文档** - 技术相关的详细文档
- **项目笔记** - 项目相关的记录和总结

## 特性

- 🚀 快速构建和部署
- 📱 响应式设计，支持移动设备
- 🔍 内置搜索功能
- 📊 支持 Mermaid 图表
- 🌍 多语言支持
