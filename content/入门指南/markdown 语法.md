# Markdown 语法

本文档使用 Markdown 语法编写。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 文本格式

- **粗体**：`**文本**`
- *斜体*：`*文本*`
- ~~删除线~~：`~~文本~~`
- `行内代码`：`` `代码` ``

## 代码块

```python
def hello():
    print("Hello, World!")
```

## 列表

### 无序列表
- 项目 1
- 项目 2
- 项目 3

### 有序列表
1. 第一步
2. 第二步
3. 第三步

## 链接和图片

- 链接：`[文本](URL)`
- 图片：`![描述](URL)`

## 表格

| 列 1 | 列 2 | 列 3 |
|------|------|------|
| A    | B    | C    |
| D    | E    | F    |

## 提示框

Docdock 主题支持提示框：

{{% notice note %}}
这是一个注释框
{{% /notice %}}

{{% notice tip %}}
这是一个提示框
{{% /notice %}}

{{% notice info %}}
这是一个信息框
{{% /notice %}}

{{% notice warning %}}
这是一个警告框
{{% /notice %}}
